# 4 Лр - Генетический алгоритм

### Задача

1. Дана строка произвольной длины N. При помощи генетического алгоритма преобразовать случайно созданную строку длины N в исходную. Отобразить полученное кол-во поколений при выбранном параметре количесва мутаций для одного потомка.
2. Дан набор строк произвольных длин N[i]. При помощи генетического алгоритма преобразовать случайно созданные строки длин N[i] в исходные. Сравнить результаты при паарметре кол-ва мутаций.
